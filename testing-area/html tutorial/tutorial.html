What is HTML?

HTML is the standard markup language for documents designed to be displayed in a web browser. A markup language is a text-encoding system which specifies the structure and formatting of a document. In layman's terms, it acts like the skeleton of a web page by telling the browser how to display the content. HTML markup consists of several components called tags. Attributes contain extra information about the element, it won't appear in the actual content. They always have a space between it and the element name, followed by an equal sign and the value is wrapped by opening and closing quotation marks.
