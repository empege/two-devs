What is HTML?

HTML is code that's used to create and structure web pages and their content. It consists of elements, for example:

The element is comprised of the opening and closing tags. The content of the element is everything between the opening and closing tag. Elements can also have attributes:

Attributes contain extra information about the element, it won't appear in the actual content. They always have a space between it and the element name, followed by an equal sign and the value is wrapped by opening and closing quotation marks.
